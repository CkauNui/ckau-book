EN:
Moved all addons outside the theme folder, in order to preserve addon files when updating theme through EmulationStation.

"/themes/ckau-book" - theme directory
"/themes/ckau-book-addons" - addons directory, which will not be overwritten by ES update procedure.

RU:
Все аддоны перемещены за пределы папки темы, чтобы сохранить файлы аддонов при обновлении темы через EmulationStation.

"/themes/ckau-book" - папка темы
"/themes/ckau-book-addons" - папка аддонов, которая не будет перезаписана при обновлении темы через ES.


== Install / Установка ==
EN: place "ckau-book-addons" folder to \themes\ (outside "ckau-book" folder)
RU: папку "ckau-book-addons" кинуть в \themes\ (рядом с папкой "ckau-book")
